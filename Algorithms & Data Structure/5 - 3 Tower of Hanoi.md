```python
# 하노이의 탑 구현하기

def move(no: int, x: int, y: int) -> None:
    if no > 1:
        move(no - 1, x, 6 - x - y)

    print(f'원반 [{no}]을 {x}기둥에서 {y}기둥으로 옮깁니다.')

    if no > 1:
        move(no - 1, 6 - x - y, y)

print('하노이의 탑을 구현합니다.')
n = int(input('원반의 개수를 입력하세요.: '))

move(n, 1, 3)
```

원반 [2]을 2기둥에서 3기둥으로 옮깁니다.
